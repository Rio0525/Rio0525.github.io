  • Sugo: Fight the Demonblight - Not Publish

    ❮ ❯


    Description

    A 3D mobile puzzle game for android that uses the concept of Sokoban or Push the Box and with a mix of dungeon crawling and was made using Unity for a capstone project.

    Software

    • Unity – is a software that allows developers to accomplish different types of tasks related to the game production process.
    • Blender – is a free open-source 3D creation suite it is used for modeling, rigging, animation, simulation, rendering, compositing, motion tracking, even video editing, and game creation
    • Free SFX – a website that provides free sound effects that can be used in a commercial, non-commercial, broadcast, multimedia/audiovisual production.
    • Paint Tool SAI – is a painting software that provides easy and stable operation that makes digital art more comfortable.
